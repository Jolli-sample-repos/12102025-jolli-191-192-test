# Authentication

Secure your application with our authentication system.

## OAuth 2.0

We support OAuth 2.0 for secure authentication.

### Supported Providers

- Google
- GitHub
- Microsoft

## API Keys

For server-to-server communication, use API keys.

```javascript
const client = new Client({
  apiKey: process.env.API_KEY
});
```
