# Configuration Guide

Learn how to configure the application for your needs.

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| PORT | Server port | 3000 |
| DEBUG | Enable debug mode | false |

## Config File

Create a `config.json` file in your project root:

```json
{
  "port": 8080,
  "debug": true
}
```
